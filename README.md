### 🐍 Python: `port_scanner.py`
High-performance TCP port scanner written in Python with:
- ⚙️ Thread pool concurrency for speed
- 🧠 Smart timeouts and graceful shutdown
- 🏷️ Banner grabbing for service fingerprinting
- 🎯 Top-N ports or custom ranges
- 🌐 DNS resolution with validation
- 🖨️ Clean, colorized output with summaries

#### 🚀 Quick Start
```
bash
# Clone
git clone https://github.com/SolvyrEryx/CyberGuardians.git
cd CyberGuardians

# Run (top 100 ports)
python port_scanner.py --target example.com --top 100 --threads 200 --timeout 0.5

# Run (custom range)
python port_scanner.py --target 192.168.1.1 --ports 1-1024 --threads 300
```

### 🛰️ New: Live Network Packet Analyzer — `packet_analyzer.py`
Real-time HTTP traffic monitor using Scapy. Features:
- 📡 Live capture on selected interface
- 🔎 Extracts and prints HTTP requests (host, path, method)
- 🧰 Filters for TCP/80 by default (customizable)
- 🛡️ Safe interrupt handling (Ctrl+C) with capture summary

Quick start:
```
python packet_analyzer.py --iface eth0 --filter "tcp port 80"
```

Note: Requires elevated privileges to capture packets on most systems.
